# Security policy — hudu-magic

This document applies to the **hudu-magic** Python client library (this repository), not the Hudu hosted product.

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a vulnerability in this library

If you believe you have found a security issue in **hudu-magic** (e.g. credential handling, unsafe defaults in the client), please **open a private security advisory** on this GitHub repository (Security → Advisories) or contact the maintainers with details and reproduction steps. We will treat valid reports seriously and aim to respond in a reasonable timeframe.

## Hudu platform

If your finding is about the **Hudu application or infrastructure** (not this Python package), use Hudu’s disclosure program: [Hudu Vulnerability Disclosure Program](https://support.hudu.com/hc/en-us/articles/11415175749271-Hudu-Vulnerability-Disclosure-Program).
