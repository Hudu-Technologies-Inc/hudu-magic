## Reporting a Vulnerability

At Hudu, we love working with security researchers.

If you are a security researcher and believe you have found a potential security issue in the Hudu platform, please contact us. We will make every effort to quickly resolve the problem.
 

## Submitting a Report
See our [Disclosure Policy and Program Rules
](https://support.hudu.com/hc/en-us/articles/11415175749271-Hudu-Vulnerability-Disclosure-Program)

After creating a report, please send it to us [here](https://support.hudu.com/hc/en-us/requests/new). Please read the following rules, policies, and information on response targets to make sure you are eligible for a reward.
